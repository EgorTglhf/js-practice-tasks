Hi there! This page is about solution of task to find a pair with given Sum exists in Sorted Array.

Given a sorted array A[] of n numbers and another number x, the task is to find two elements in A[] whose sum is exactly x.

Input: arr[] = {-3, -1, 0, 2, 1}, x= -2
Output: {-3, 1}
Explanation: If we calculate the sum of the output,1 + (-3) = -2
